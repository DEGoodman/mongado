## Quick Reference

### Backend Commands

```bash
cd backend
make test       # Run tests
make ci         # Full CI pipeline
make profile    # Profile performance
```

See [TESTING.md](TESTING.md) for more.

### Frontend Commands

```bash
cd frontend
npm test        # Run tests
npm run test:all  # Full test suite
npm run build:analyze  # Bundle analysis
```

See [TESTING.md](TESTING.md) for more.
